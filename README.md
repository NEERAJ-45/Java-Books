## **Phase 2: Distributed Systems & Databases (Years 2-5)**

| **Book** | **Why** |
| --- | --- |
| **Designing Data-Intensive Applications** (Martin Kleppmann) | Replication, partitioning, transactions, consistency – non-negotiable |
| **Database Internals** (Alex Petrov) | MVCC, B-trees, LSM trees, isolation levels – beyond theory |
| **Building Microservices** (Sam Newman) | Microservices architecture, Saga patterns, distributed transactions |
| **Kafka: The Definitive Guide** (Confluent) | Event streaming, exactly-once semantics, partitioning |
